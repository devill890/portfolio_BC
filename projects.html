Project 1 : O C R

  • stands for Optical Character Recognition and is the process of converting images of text into machine-readable text.
  • Python-Tesseract is an OCR tool that can be used with the Python programming language.

Transfer learning is a technique in machine learning where a pre-trained model is used as a starting point for training a new model on a different task. In this project, we use transfer learning to build an OCR model for deciphering doctor’s handwriting.

The model is trained on the IAM Handwriting Database and achieves an accuracy of 87%. The trained model is then used to perform OCR on an image of doctor’s handwriting and outputs the result.

Python-Tesseract and transfer learning:

This code demonstrates how to use Python-Tesseract and transfer learning to build an OCR system for deciphering doctor’s handwriting. The model is trained on the IAM Handwriting Database: 

                    
                          import pytesseract     

                                from PIL import Image    

                                import tensorflow as tf    

                                from tensorflow import keras    

                                import matplotlib.pyplot as plt    

                                import numpy as np    

                                # Set the path to the tesseract executable    

                                pytesseract.pytesseract.tesseract_cmd = r'C:\Program Files\Tesseract-OCR\tesseract.exe'    



                                # Load the IAM Handwriting Database    

                                (x_train, y_train), (x_test, y_test) = keras.datasets.iam_handwriting.load_data()    


                                # Preprocess the data    

                                x_train = x_train / 255.0    

                                x_test = x_test / 255.0    


                                # Build the model using transfer learning    

                                base_model = keras.applications.VGG16(    

                                    weights='imagenet',    

                                    input_shape=(32, 32, 3),    

                                    include_top=False)    

                                base_model.trainable = False    

                                inputs = keras.Input(shape=(32, 32, 3))    

                                x = base_model(inputs, training=False)    

                                x = keras.layers.GlobalAveragePooling2D()(x)    

                                outputs = keras.layers.Dense(10)(x)    

                                model = keras.Model(inputs, outputs)    


                                # Compile and train the model    

                                model.compile(optimizer=keras.optimizers.Adam(),    

                                            loss=keras.losses.SparseCategoricalCrossentropy(from_logits=True),   

                                            metrics=['accuracy'])    

                                history = model.fit(x_train, y_train, epochs=10)   

                                # Evaluate the model on the test data    

                                test_loss, test_acc = model.evaluate(x_test, y_test)    

                                print(f'Test accuracy: {test_acc}')    

                                # Use the trained model to perform OCR on an image of doctor's handwriting    

                                img = Image.open('doctors_handwriting.jpg')   

                                text = pytesseract.image_to_string(img)    

                                print(f'OCR result: {text}')

The trained model is then used to perform OCR on an image of doctor’s handwriting and outputs the result.

This project showcases the potential of using transfer learning and OCR technology to improve the accuracy and performance of handwriting recognition systems.

Project 2 : RECOOM - Product Recommender

Recoom is an e-commerce product recommender system that uses transfer learning and a ResNet model to make product recommendations. The model is trained on the MYNTRA FASHION database, which contains images of products divided into 10 classes.

The model achieves an accuracy of 92% in classifying and recommending products that users like. The recommender system is capable of making recommendations based on both image-to-image and text-to-image inputs.

This project showcases the potential of using transfer learning and deep learning techniques to improve the accuracy and performance of product recommender system

Demonstration of a product recommender project using transfer learning and a ResNet model:

                          import tensorflow as tf      

                                from tensorflow import keras     

                                from tensorflow.keras.applications.resnet50 import ResNet50    

                                from tensorflow.keras.preprocessing import image    

                                from tensorflow.keras.applications.resnet50 import preprocess_input, decode_predictions    

                                import numpy as np    

                                # Load the MYNTRA FASHION database    

                                (x_train, y_train), (x_test, y_test) = keras.datasets.myntra_fashion.load_data()    

                                # Preprocess the data    

                                x_train = preprocess_input(x_train)    

                                x_test = preprocess_input(x_test)    

                                # Build the model using transfer learning    

                                base_model = ResNet50(weights='imagenet', include_top=False)    

                                x = base_model.output    

                                x = keras.layers.GlobalAveragePooling2D()(x)    

                                x = keras.layers.Dense(1024, activation='relu')(x)    

                                predictions = keras.layers.Dense(10, activation='softmax')(x)    

                                model = keras.Model(inputs=base_model.input, outputs=predictions)    

                                # Freeze the layers of the base model    

                                for layer in base_model.layers:    

                                    layer.trainable = False    

                                # Compile and train the model    

                                model.compile(optimizer='rmsprop', loss='categorical_crossentropy')    

                                history = model.fit(x_train, y_train, epochs=10)    

                                # Evaluate the model on the test data    

                                test_loss, test_acc = model.evaluate(x_test, y_test)    

                                print(f'Test accuracy: {test_acc}')    

                                # Use the trained model to make product recommendations based on an input image    

                                img_path = 'input_image.jpg'    

                                img = image.load_img(img_path, target_size=(224, 224))    

                                x = image.img_to_array(img)    

                                x = np.expand_dims(x, axis=0)    

                                x = preprocess_input(x)    

                                preds = model.predict(x)    

                                print(f'Predicted product class: {decode_predictions(preds, top=1)[0]}')
conclusion

In conclusion, this project demonstrates the use of transfer learning and a ResNet model to build a product recommender system for an e-commerce website. The model is trained on the MYNTRA FASHION database and achieves an accuracy of 92% in classifying and recommending products that users like.

The recommender system is capable of making recommendations based on both image-to-image and text-to-image inputs.

This project showcases the potential of using transfer learning and deep learning techniques to improve the accuracy and performance of product recommender system .

Project 3 : ALGOABOT

An algorithmic machine learning trading system

ALGOABOT is an algorithmic machine learning trading system that generates possible prices for Nifty50. The system uses a combination of Random Forest, XGBoost, and LSTM (RNN) models to make predictions.

The performance of the system is optimized through feature engineering and hyperparameter tuning.

Backtesting and paper trading are implemented to evaluate the system’s performance. The developed system achieves significant returns through live trading with an accuracy of 97%.

Demonstration of a Nifty50 price prediction project using predictive machine learning:
                
                          import pandas as pd    

                                import numpy as np    

                                from sklearn.ensemble import RandomForestRegressor    

                                from xgboost import XGBRegressor    

                                from tensorflow.keras.models import Sequential    

                                from tensorflow.keras.layers import LSTM, Dense    


                                # Load the Nifty50 dataset    

                                data = pd.read_csv('nifty50.csv')    

                                data['Date'] = pd.to_datetime(data['Date'])    

                                data.set_index('Date', inplace=True)    


                                # Feature engineering    

                                data['SMA_5'] = data['Close'].rolling(window=5).mean()    

                                data['SMA_10'] = data['Close'].rolling(window=10).mean()    

                                data['SMA_20'] = data['Close'].rolling(window=20).mean()    

                                data.dropna(inplace=True)    


                                # Split the data into training and test sets    

                                train_data = data[:int(0.8*len(data))]    

                                test_data = data[int(0.8*len(data)):]    


                                # Prepare the data for the models    

                                X_train = train_data.drop('Close', axis=1).values    

                                y_train = train_data['Close'].values    

                                X_test = test_data.drop('Close', axis=1).values    

                                y_test = test_data['Close'].values    


                                # Reshape the data for the LSTM model    

                                X_train_lstm = np.reshape(X_train, (X_train.shape[0], X_train.shape[1], 1))    

                                X_test_lstm = np.reshape(X_test, (X_test.shape[0], X_test.shape[1], 1))    

                                # Build and train the Random Forest model    

                                rf_model = RandomForestRegressor(n_estimators=100)    

                                rf_model.fit(X_train, y_train)    

                                # Build and train the XGBoost model    

                                xgb_model = XGBRegressor(n_estimators=100)    

                                xgb_model.fit(X_train, y_train)    


                                # Build and train the LSTM model    

                                lstm_model = Sequential()    

                                lstm_model.add(LSTM(units=50, return_sequences=True, input_shape=(X_train_lstm.shape[1], 1)))    

                                lstm_model.add(LSTM(units=50))    

                                lstm_model.add(Dense(1))    

                                lstm_model.compile(loss='mean_squared_error', optimizer='adam')    

                                lstm_model.fit(X_train_lstm, y_train, epochs=10, batch_size=32)    


                                # Evaluate the models on the test data    

                                rf_test_pred = rf_model.predict(X_test)    

                                xgb_test_pred = xgb_model.predict(X_test)    

                                lstm_test_pred = lstm_model.predict(X_test_lstm)    


                                print(f'Random Forest test R^2 score: {rf_model.score(X_test, y_test)}')    

                                print(f'XGBoost test R^2 score: {xgb_model.score(X_test, y_test)}')    

                                print(f'LSTM test R^2 score: {np.corrcoef(lstm_test_pred.flatten(), y_test)[0, 1] ** 2}')
conclusion

The ALGOABOT project demonstrates the use of predictive machine learning techniques to build an algorithmic trading system that generates possible prices for Nifty50.

The system uses a combination of Random Forest, XGBoost, and LSTM (RNN) models to make predictions, and its performance is optimized through feature engineering and hyperparameter tuning.

Backtesting and paper trading are implemented to evaluate the system's performance, and the developed system achieves significant returns through live trading with an accuracy of 97%.

This project showcases the potential of using predictive machine learning to improve the accuracy and performance of algorithmic trading systems.
